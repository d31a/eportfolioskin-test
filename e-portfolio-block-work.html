React Calculator App

Demonstrating State


Skills Used:

HTML/CSS React UI/UX

This is an improvement of the Meta React Basics course calculator created as a final project. Although this calculator was functional, it wasn't very efficient, so the challenge was to create a better functioning calculator in React. The end result is a calculator that creates calculations by storing the properties of the previous value, current value and operator values using the React state hook. The logic flow was co-ordinated using a logic flow diagram, and then executed.


These are the achieved requirements:

- Create an improved & working React calculator app

- Demonstrate React state

- Show clearly the logic flow of an app

- Demonstrate a user-friendly app with CSS application

- Use HTML/CSS and React JS to create a functioning app

- Demonstrate how a component can be created and referenced
